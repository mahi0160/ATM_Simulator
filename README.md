ATM Simulator (Java)

Description

This project is an ATM Simulator developed using Java. It simulates basic ATM operations such as account access, balance checking, deposit, and withdrawal.

Features

- User login with PIN
- Check account balance
- Deposit money
- Withdraw money
- Exit system

Technologies Used
- Java
- VS Code / Any Java IDE

How to Run

1. Open the project in VS Code or any Java IDE
2. Compile the file:
javac ATMSimulator.java
3. Run the program:
java ATMSimulator

Project Overview

This program is designed to demonstrate the working process of an ATM system using basic programming concepts like loops, conditions, and functions.
